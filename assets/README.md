# Assets

No logo, icon set, or illustration files were provided with this brief — it was written from scratch
(see readme.md "Source of this design system"). Per design-system policy, no icons or illustrations
were hand-drawn or generated to fill the gap.

**Icons:** [Material Symbols Rounded](https://fonts.google.com/icons), loaded from Google's font CDN
in every component/card/UI-kit HTML file:

```html
<link rel="stylesheet" href="https://fonts.googleapis.com/css2?family=Material+Symbols+Rounded:opsz,wght,FILL,GRAD@24,400,0,0">
<span class="material-symbols-rounded">nutrition</span>
```

Rounded style chosen to match the Baloo 2 / Nunito rounded type direction and the app's
Android/Material-3 sensibility.

**Illustrations/photography:** none supplied. Splash, onboarding and empty states use flat-color
panels with a centered Material Symbol as a structural placeholder — see readme.md Iconography
section for the full explanation and the ask to the user.
